# AGENTIC AI + RAG MASTERY ROADMAP

This workspace contains multiple projects focused on agentic AI, retrieval-augmented generation (RAG), FastAPI applications, and machine learning foundations.

## Main folders

- `agentic-ai/` — Core RAG and agentic AI projects, with phase-based prototypes.
- `phase 1 - python/` — Python practice and FastAPI sample projects.
- `phase2 - LLMFoundations/` — LLM fundamentals, hyperparameters, and token metrics.
- `Phase - Machine Learning/Scikit-learn/` — Machine learning practice using scikit-learn.

## Key projects

### agentic-ai/

- `phase2/fastapi-agentic-rag-Project/` — FastAPI + LangChain RAG application using PDF retrieval and Chroma.
- `phase3/production-rag/` — Production-ready RAG app variant with similar architecture.
- `phase1/` — Earlier prototype and API examples for agentic systems.

### phase 1 - python/

- FastAPI sample endpoints and basic Python learning scripts.

### phase2 - LLMFoundations/

- Exploratory code for LLM hyperparameters and token metrics.

### Phase - Machine Learning/Scikit-learn/

- Linear regression examples and notes for supervised learning practice.

## Getting started

1. Choose the folder for the project you want to work on.
2. Read the `README.md` file inside that folder for specific installation and usage instructions.
3. Use a Python virtual environment and install dependencies as needed.

## Notes

- Some folders already contain `requirements.txt`, `projectstructure.txt`, or sample application code.
- Sensitive credentials (for example, PostgreSQL connection details) should be moved to `.env` before sharing or deploying.
