# Scikit-learn Practice

This folder contains machine learning practice scripts using scikit-learn, with a focus on linear regression.

## Contents

- `Linear Regression.py` — Linear regression examples and training/prediction exercises.
- `notes.txt` — Notes on linear regression, context windows, and LLM prompt ideas.
- `rough.py` — Additional experimentation or rough work.

## Purpose

- Practice supervised learning with `LinearRegression`.
- Learn how to fit models and make predictions.
- Explore simple regression workflows in Python.

## Running the example

1. Create and activate a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
```

2. Install dependencies:

```bash
pip install scikit-learn numpy
```

3. Run the script:

```bash
python "Linear Regression.py"
```

## Notes

- This folder is a learning space for model training and evaluation.
- Use the script to experiment with different linear relationships and predictions.
